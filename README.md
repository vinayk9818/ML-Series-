#  Machine Learning Masterclass: Core Concepts & Notes

Welcome to my personal Machine Learning learning repository! This project is a curated series of comprehensive notes, hands-on implementations, and cheatsheets covering everything from foundational statistics to advanced ML algorithms. 

Whether you are a beginner looking to understand the math behind the models or an advanced practitioner revising core concepts, this repository serves as a structured roadmap.

---

##  Series Overview

The goal of this series is to break down complex ML algorithms into intuitive concepts, back them up with mathematical formulations, and implement them using industry-standard Python libraries.

### What's Inside:
*   **Theoretical Deep-Dives:** Clear explanations of the "why" and "how" behind algorithms.
*   **Math Formulations:** Breaking down the equations (loss functions, optimization, etc.) without the intimidation.
*   **Code Implementation:** Clean, documented Jupyter Notebooks using `NumPy`, `Pandas`, `Scikit-Learn`, and more.
*   **Interview Preparation:** Key conceptual questions and quick-revision cheatsheets.

---

##  Roadmap & Topics Covered

###  Phase 1: Foundations & Classical ML
-  1. **Introduction to ML:** Supervised vs. Unsupervised, Bias-Variance Tradeoff.
-  2. **Linear Regression:** Cost Function, Gradient Descent, Evaluation Metrics ($R^2$, MSE).
- 3. **Logistic Regression:** Sigmoid Function, Binary Cross-Entropy, Decision Boundaries.
- 4. **Regularization:** Lasso (L1) and Ridge (L2) Regression *(In Progress)*

### 🔹 Phase 2: Tree-Based Models & Ensemble Learning
- [ ] **Decision Trees:** Information Gain, Gini Impurity, Entropy.
- [ ] **Ensemble Methods:** Bagging vs. Boosting.
- [ ] **Random Forest & XGBoost:** Hyperparameter tuning and handling feature importance.

### 🔹 Phase 3: Unsupervised Learning & Dimensionality Reduction
- [ ] **Clustering:** K-Means, Hierarchical Clustering, Elbow Method.
- [ ] **Dimensionality Reduction:** Principal Component Analysis (PCA) intuition and math.
